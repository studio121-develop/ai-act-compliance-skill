## How to Maintain

1. **New AI system**: Add a row to the system table. Run the risk classification decision tree and compliance score card.
2. **Quarterly review**: Update scores, check for new blockers, verify documentation is current.
3. **New AI feature in existing system**: Re-classify. Update the dashboard if classification changes.
4. **Regulatory update**: After running the skill's auto-update workflow, review all systems for impact.
5. **Incident**: Log it, update the relevant system's status, track remediation.
